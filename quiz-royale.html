Quiz Royale

Battle Royale, aber als Quizshow im Browser - Spieler treten in Echtzeit gegeneinander an.

GitHub ansehen ← Zurück

Wie ist das entstanden?

Quiz Royale war ein Semesterprojekt im Rahmen der Kurse Ubiquitous Computing und Audiovisuelle Umgebungen an der Hochschule Furtwangen. Die Idee: ein Multiplayer-Quizspiel komplett im Browser, ohne externe Game-Engine – nur mit Web-Technologien.

Eine der Herausforderungen war das faire Frage-Shuffling: Alle Spieler sollen dieselben Fragen in derselben Reihenfolge sehen, ohne dass der Server jede einzeln schickt. Lösung: Seed-basiertes Shuffling.

Technologien

  • Three.js & WebGL / GLSL
  • WebSockets (Echtzeit-Multiplayer)
  • JavaScript (Vanilla)
  • Seed-basiertes Frage-Shuffling
  • GPU-beschleunigte Partikeleffekte

Entstehung

  • Kurs: Ubiquitous Computing & AV-Umgebungen
  • Hochschule Furtwangen
  • Synchronisation der Mitspieler:innen
  • 6-minütige Pitch-Präsentation
